# u8g2 for Cortex-M3 Corte-M4

这是南京邮电大学集成电路科学与工程学院学院科学与技术协会学长编译的支持cortex-m3 cortex-m4内核的u8g2静态链接库  
* 只保留了ssd1306的驱动（在一定程度上也可用于ssd1315）
* 去除了mui
* 包含所有内置字体。
* 编写了适合STM32平台 ( HAL ) 的接口文件

## 如何选择适合的版本
* 库文件方面，如果使用keil开发，选择u8g2-lib-cortex-m3/4-keil（wchar16版本）
* 如果使用STM32CubeIDE、STM32CubeCLT等使用gcc的工具链，选择gcc（wchar32版本）
* 实际上，keil(默认使用Arm Compiler，也可以切换gcc)，也可以用wchar32，gcc也可以用wchar16，只是由于默认值的不同，需要额外编译参数指定（对于新生，这必然是翻车重灾区），因此这里分别编译了两种版本
* 此外，keil会把.a后缀的静态库误识别为汇编，需要手动修改，因此keil版本的库重命名为了.lib后缀（其实这个版本的库也是gcc编译的）
* F1系列选择m3，F4系列选择m4（理论上，只要内核相符即可）

## 如何使用
* 把 libu8g2.a (libu8g2.lib) 作为静态链接库添加到你的项目中
* 把 对应的port文件夹下的文件：.h 添加到 Core/Inc .c 添加到Core/Src !!!并作为源文件添加到你的项目中!!!
* 把 u8g2_include 中的头文件添加到 Core/Inc
* 虽然文件路径不是死的，但是对于不了解工具链行为的新人，强烈建议你一步一步跟着走
* 初始化I2C外设，传入 hi2c 句柄进行初始化
* 学会看例程

## 示例
* Tutorial_u8g2_F1是F103的keil示例，Tutorial_u8g2_F4 是F401的例程，由于本仓库主要目的是帮助学生学习、参与竞赛和完成作品，目前仅制作了keil的例程
